## ChitChat > Java Client

### Setup

 1. Run the following to set-up Gradle with IntelliJ IDEA
```
gradle cleanIdea
gradle idea
```
 2. Open `./ChitChatDesktop` in IntelliJ IDEA.


### Test Execution

#### Invoke Tasks

```
invoke test
```



#### E2E Tests

You can run:

```
cd e2e
invoke test
```

The video output will be at:
```
ChitChatDesktop/ChitChatDesktop.mp4
```
